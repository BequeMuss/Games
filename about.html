About This Project

The project is non-commercial, contains no advertising, and is not affiliated with any company or brand. It is intended purely for educational purposes and as a demonstration of my programming progress.

All code and content were developed independently. The site does not include user accounts, tracking tools, or third-party services.

I hope that this project can be accessed in educational environments, as it was built to encourage learning and creativity through coding.
